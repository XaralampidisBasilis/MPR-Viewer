Gestures

Desktop

1 / 2 / 3 / 4: switch to Place / Inspect / Edit / Segment

right mouse drag: orbit camera

mouse wheel: zoom camera

ctrl+z / ctrl+y: undo / redo

x: toggle brush add/subtract in Edit and Segment

c: clear Segment prompt points

General

swipe down: undo

swipe up: redo

swipe right/left: change mode

implode: reset

explode: exit AR

Place

desktop left drag: move volume

desktop alt + left drag: rotate volume

desktop shift + mouse wheel: scale volume

double tap: place volume on cursor/hide volume

pan: rotate volume

pinch: scale volume

twist: roll volume

hold: move volume

Inspect

desktop left drag on slice: move slice

desktop alt + left drag on slice: rotate slice stack

desktop double click on slice: hide/reveal plane

pan: rotate planes

twist: rotate all 3 planes

hold: move planes

hold center: move all 3 planes

double tap: hide/reveal plane

pinch: scale volume

Edit

desktop left drag on slice: paint mask

desktop shift + mouse wheel: resize brush

desktop double click or x: toggle brush mode

pan: rotate volume

twist: change image contrast

hold: edit mask

double tap: change brush mode

pinch: scale volume

Segment

desktop left click on slice: add prompt point

desktop alt + mouse wheel: move current slice

desktop shift + mouse wheel: resize brush

desktop double click or x: toggle prompt label
